# Latte

## Kompilacja
Program kompilujemy przy użyciu polecenia `make`

## Użyte biblioteki
- base
- containers
- array
- mtl
- lens
- extra

## Struktura
- src/main - pliki main
- src/latte/Frontend - implementacja frontendu
- src/parser - pliki parsera

## Składnia
Składnia rzutowania różni ma postać `"null" "as" Typ`, przykładowo `null as list`.
